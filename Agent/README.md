# Agent Workspace

This folder carries agent guidance and local context for this submodule.

## Folder Intent

- `Standard/`: submodule guidance and relative links to the containing product repository's canonical standards.
- `Initiative/`: active submodule-specific plans, checklists, progress notes, and decision records.
- `Archive/`: superseded or legacy agent documents kept for historical context.

## Working Rule

When work in a containing product repository changes this submodule, record the relevant submodule-owned plan, checklist, decision, or archive note here so future consumers can understand the change without reading the containing product repository history.
